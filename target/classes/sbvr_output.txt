Es ist notwendig, dass der Prozess mit dem Startevent Signed employment 
contract on file startet. 
DataInput 'employment contract' ist Eingabe für Aktivität 'Add personal data'.
DataObject 'Work equipment' ist Ausgabe von Aktivität 'Select necessary work equipment'.
 Es ist notwendig, dass die Aktivität 'Procure work equipment' das Datenobject'Work equipment' erzeugt.
Es ist notwendig, dass die Aktivität Procure work equipment eine Nachricht von Supplier empfängt, bevor fortgeführt wird.
Es ist notwendig, dass die Aktivität Procure work equipment eine Nachricht an Supplier sendet, bevor fortgeführt wird.
„Es ist notwendig, dass der Subprozess Procure work equipment die Nachricht [Unbekannte Nachricht] vom Supplier empfängt.“
„Es ist notwendig, dass der Subprozess Procure work equipment die Nachricht [Unbekannte Nachricht] an Supplier sendet.“
Es ist notwendig, dass der Prozess mit dem EndEvent "Prepared for
 employee to start" endet, nachdem die Aktivität "unbenannte Aktivität" abgeschlossen wurde.
